# **App Name**: Narrative Wellness Engine (NWE)

## Core Features:

- Personalized Health Narrative Generation: Generates a personalized health narrative based on user data, providing proactive guidance, reducing uncertainty, and simplifying chronic care management. It will function as a tool for synthesizing data, so the LLM can make informed choices.
- Predictive Environment Generation (PEG): Generates health alerts and proactive plans based on environmental factors like air quality, pollen, and temperature.
- Generative Companion Prompt: Provides motivational support and coaching through a persistent chat bubble with a coaching persona.
- Longitudinal Data Analysis and Visualization: Synthesizes longitudinal data to identify trends and correlations, providing insights into the user's health.  It will function as a tool for identifying key pieces of information from long time frames to highlight for the user.
- Generative Self-Management Tips: Provides self-management tips triggered by the interactive body map, tailoring advice based on selected pain areas.
- Generative Summary for Provider (GSP): Generates shareable health summaries for providers, solving the problem of inefficient clinical visits. It will function as a tool, identifying pivotal events to include.
- Personalized Healthcare News Synthesis: Delivers a personalized wellness digest with GenAI-filtered healthcare news.

## Style Guidelines:

- Primary color: Vibrant Teal/Aqua (#008080) for CTAs and success indicators, representing health/flow.
- Background color: Desaturated light-brown (#F0EAD6), influenced by terracotta.
- Accent color: Savannah green (#8FBC8F), approximately 30 degrees from the primary hue on the color wheel.
- Body and headline font: 'Comfortaa' (sans-serif) for all text.
- Clean and minimalist layout with widgets using rounded corners and shadows, organized in a clear, scrollable hierarchy.
- Smooth transitions and engaging micro-animations for onboarding and data updates on the dashboard.
- Simple, clear icons to represent various features and data points. Use of data visualization (charts, body maps) and motion to demonstrate intelligence.
